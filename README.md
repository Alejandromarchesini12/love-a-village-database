# Love a Village Database System

A relational database for the “Love a Village” NGO in Malawi, designed to manage donors, donations, projects, volunteers, expertise, and village requests.

## Files
- `Love_A_Village.sql` — Full database export (structure & data)
- `DDL - LoveAVillage.sql` — Schema (tables, constraints)
- `DML - LoveAVillage.sql` — Sample data
- `function_test_cases.sql` — Test cases for stored functions
- `procedures.sql` — Stored procedures
- `views.sql` — Created views
- `EER_Diagram.pdf` — Entity-Relationship diagram

## Project Highlights
- Tracks donors, donations, projects, volunteers (experts & non-experts), expertise, village requests, and assignments for a real-world nonprofit.
- Enforces business rules and referential integrity with constraints and foreign keys.
- Supports advanced queries and analytics with provided test cases, stored procedures, and views.
- Includes a full EER diagram and documentation for easy understanding and extension.
- Designed for scalability and clarity, making it easy to adapt for other NGOs or similar organizations.

## About
Built for a non-profit supporting villages in Malawi, this system helps match resources, people, and funding to community needs.
